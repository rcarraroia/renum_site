# Frontend - Renum Platform

Aplicação React com Vite, TypeScript e Tailwind CSS.

## Desenvolvimento

```bash
npm install
npm run dev
```

O frontend rodará em `http://localhost:5173`

## Build

```bash
npm run build
```

## Deploy

Automaticamente via Vercel quando push na branch `main`.

## Estrutura

- `src/` - Código fonte
- `public/` - Assets estáticos
- `src/components/` - Componentes React
- `src/pages/` - Páginas da aplicação
- `src/context/` - Contextos React
- `src/lib/` - Bibliotecas e utilitários

## Tecnologias

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui
- React Router
- Supabase Client
